# Strategic Miscalibration
This repository serves as a research framework for studying strategic miscalibration in LLMs. It implements a repeated signaling game with reputation, elicits strategies and gameplay from LLLMs, and analyzes the resultant behavior and outcomes. 

## Dependencies
LLM queries are handled via [LiteLLM](https://docs.litellm.ai/docs/) and [Together AI](https://docs.together.ai/intro).
